# Kaite Bloom Co., Ltd. — Website

Single-page bilingual marketing site for **Kaite Bloom Co., Ltd.**, a Koh Phangan–based real estate operations company powered by WhatsApp-native AI.

**Live:** https://liam-kp.github.io/kaite-bloom-ai/

## Stack
- HTML + Tailwind (CDN, no build step)
- Vanilla JS for the EN/TH language toggle
- Hosted on GitHub Pages from `main` branch root

## Files
- `index.html` — main site
- `404.html` — fallback page
- `assets/img/` — optimized images
- `assets/demo-villa-th.html` — embedded live demo
- `robots.txt`, `sitemap.xml`, `.nojekyll` — SEO + Pages config

## Edit
Open `index.html` directly. To update copy, edit the visible HTML AND the matching key in the `I18N` object inside the `<script>` block at the bottom.
